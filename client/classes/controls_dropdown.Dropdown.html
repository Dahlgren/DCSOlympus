setOptions

  • setOptions(optionsList, sort?, labelsList?): void

  • Set the dropdown options strings

    Parameters

    • optionsList: string[]

      List of options. These are the keys that will always be returned on selection

    • sort:
          | null
          | "string"
          | "number"
          | "string+number" = "string"

      Sort method. null means no sorting. "string" performs js default sort. "number" sorts purely by numeric value. "string+number" sorts by string, unless two elements are lexicographically identical up to a numeric value (e.g. "SA-2" and "SA-3"), in which case it sorts by number.

    • labelsList: undefined | string[] = undefined

      (Optional) List of labels to be shown instead of the keys directly. If provided, the options will be sorted by label.

    Returns void

    • Defined in src/controls/dropdown.ts:51
